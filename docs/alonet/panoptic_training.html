Training¶

For training, LitPanopticDetr implements a Pytorch Lightning Module that uses as default the detr_r50 module coupled with detr_panoptic. For this reason, alonet.detr.criterion and alonet.detr.matcher are used in the training. However, the alonet.detr_panoptic.callbacks are adapted to the predictions of the masks in the inference process.
